This allows you to install Cygwin from a local directory instead
of downloading it. To install do the following:

1) Double Click on "setup.exe"
2) Relase Notes: Click "Next>"
3) Choose A Download Source: Select "Install from Local Directory" and then click "Next>"
4) Select Root Install: Click "Next>"
5) Select Local Package Directory: 
	- Click on "Default" written next to the "- ALL" right below Category
		- It should change to "Install"
	- Click "Next>"
6) Select Packages: Click "Next>"
7) Click "Finish"
